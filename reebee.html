Auto Delete Expired Items

Context:

reebee has a shopping list feature which lets users add flyer items for convenient access when shopping.

Problem:

When the shopping list items' flyers became expired they were no longer useful and users would complain about them accumulating in their list.

Goal:

To automate removing expired items from users shopping list.

Solution:

The "Auto Delete Expired Items" feature adds functionality for users to decide if they would like the app to delete expired itemsand at which selected time interval. It incorporates a check when the app enters the foreground to determine if there are expired items to be deleted based on users settings and will notify users if it has done so, as well as an onboarding dialog teaches the user about the feature and if they'd like to use it.

Item Options

Context:

Problem:

Many users requested the ability to add and edit notes and quantity to items in their shopping list.

Goal:

To create a quantity and notes section for items in the shopping list, and use the quantity information to calculate total price of the shopping list.
