Auto Delete Expired Items

Context:

reebee has a shopping list feature which lets users add flyer items for convenient access when shopping.

Problem:

For most users when the shopping list items' flyers became expired they were no longer useful and would accumulate in the list. The user would have to manually delete these items when they wanted them removed.

Goal:

To create a method to automate removing expired items from users shopping list.

Solution:

The "Auto Delete Expired Items" feature adds functionality for users decide if they would like the app to delete expired items for them and at which selected time interval. It incorporates a check when the application initially enters the foreground to determine if there are any items expired item to be deleted based on users settings and will notify users if it has done so, as well as an onboarding dialog teaches the user about the feature and if they'd like to use it.

Item Options

Context:

reebee has a shopping list feature which lets users add flyer items for convenient access when shopping.

Problem:

Many users requested the ability to add and edit notes and quantity to items in their shopping list.

Goal:

To create a quantity and notes section for items in the shopping list, and use the quantity information to calculate total price of the shopping list.

Solution:

The design I selected incorporates an arrow that expands and collapses the new section of an item. This dropdown contains sections to edit quantity and note for each item. When the user enters a quantity a total price for the item will be calculated. All the item totals will then be calculated and a total price of the users shopping list can be calculated. After the user is done editing these options they can collapse the section and it will only show information the user has edited.

Results:

This feature added three major changes to the shopping list, quantity, notes and being able to provide users with a total price of their shopping list. The reebee team have wanted to add this feature based on customer feedback for some time now and these three changes greatly increase the functionality of the shopping list.
