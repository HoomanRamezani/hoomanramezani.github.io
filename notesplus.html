Notes+

Welcome to Notes+!

Notes+ is an elegant notes application for iOS devices that provides users with a simple and intuitive note taking expierience. The app is quick and easy to open and allows the user to instantly begin writing whatever they desire.

Features

  • Notes can be quickly created and saved or removed on main menu
  • Notes can be given a title, description and photo
  • Photos can be attached to notes that are able to be seen from the main menu
  • Photos are chosen from camera roll or device camera
  • Visually attractive user interface

How Notes+ Was Created

Notes+ was written in Swift on XCode. The app utilized core data to implement reading and saving data with alerts to provide users with a simple and convenient experience to write notes. I had no previous expierience writing iOS applications and utilized this project to familiarize myself with app development and learning Swift.
